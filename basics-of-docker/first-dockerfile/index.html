First Dockerfile

This is a simple Dockerfile example.

Dockerfile Example

            
                FROM ubuntu:latest
                RUN apt-get update && apt-get install -y python3
                COPY . /app
                WORKDIR /app
                CMD ["python3", "app.py"]

Build and Run

To build the Docker image, run:

docker build -t my-python-app .

To run the Docker container, use:

docker run my-python-app
